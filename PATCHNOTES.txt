────────────────────────────────────────
  Phase 1 — GPT (2026-03-04 ~ 2026-03-09)
────────────────────────────────────────

[v0.1] 초기 세팅
  - Python 3.14.3 + discord.py 기반 봇 프로젝트 구성
  - 가상환경(venv) 세팅 및 필요 패키지 설치
    (discord.py, aiohttp, icalendar, python-dotenv)
  - run.bat 생성 → 더블클릭으로 봇 실행 가능하도록 구성
  - bss.env에 DISCORD_TOKEN, CHANNEL_ID 등 환경변수 관리

[v0.2] 토트넘 알림 기능
  - /bbup 커맨드: 토트넘 경기 DM 알림 구독
  - /bbdown 커맨드: 구독 해제
  - DM 발송 시점: 경기 24시간 / 1시간 / 킥오프 전
  - ICS 캘린더(토트넘 공식) 파싱으로 경기 일정 관리

[v0.3] F1 알림 기능
  - F1 ICS 캘린더 파싱 추가
  - DM 알림에 F1 경기 포함

[v0.4] 라인업 기능
  - Sofascore API로 공식 선발 라인업 조회
  - 경기 1시간 전 DM에 라인업 포함
  - lineup_loop 추가 (5분 간격 체크)

  Phase 2 — Claude (2026-03-10)

[v0.5] DM 타이밍 개선 + F1 세션 구분
  - DM 발송 시점 변경: 24시간 / 30분 / 10분 전
    (기존 24시간 / 1시간 / 킥오프 → 수정)
  - 30분 전 / 10분 전 DM에 라인업 포함
  - F1 세션 타입 구분 표시
    예: 프리 프랙티스 1 / 예선 / 스프린트 / 본경기
  - /bbf1 커맨드 추가 (GP 전체 세션 일정)
  - result_loop 추가: 경기 종료 후 결과 자동 채널 발송
  - /bbtime 커맨드 추가
  - /bbup 구독 모드 선택 추가 (토트넘만 / F1만 / 전체)

[v0.6] 멀티서버 지원 + /bbhelp
  - 단일 서버 고정 방식 → 다중 서버 지원으로 변경
  - guild_settings.json으로 서버별 채널 ID 관리
  - /bbset 커맨드: 서버별 알림 채널 지정
  - 글로벌 슬래시 커맨드 전환 (sync_commands.py로 구 명령어 초기화)
  - /bbhelp 커맨드 추가: 명령어 전체 안내

[v0.7] 봇 상태창
  - 봇 상태(presence)에 다음 일정 자동 표시
  - 표시 형식: "vs Atletico 03/11 21:00 | FP1 03/14 09:00"
  - F1 세션 종류 구분 표시

[v0.8] /bblast 시도 및 제거
  - /bblast 커맨드 추가: 최근 경기 결과 + 라인업 + MOM + 선수평점 DM 발송
  - Sofascore API 403 오류로 데이터 수신 불가 확인
  - /bblast 커맨드 제거

[v0.9] 코드 리팩터링
  - bot.py 1000줄 → 4개 파일 분리
      config.py   — 환경변수, 상수
      utils.py    — ICS/Sofascore 헬퍼, 포맷 함수
      commands.py — 슬래시 커맨드 전체
      bot.py      — 봇 진입점, 이벤트 루프
  - presence: Sofascore last 호출 제거, ICS 기반 다음 일정만 표시
  - 팀명 자동 단축 처리 (Atletico Madrid → Atletico 등)
  - PATCHNOTES.txt / README.md 추가

================================================================
  현재 커맨드 목록
================================================================
  /bbset    — 알림 채널 지정 (관리자용)
  /bbtime   — 다음 토트넘 + F1 일정 확인
  /bbf1     — F1 다음 GP 전체 세션 일정
  /bbup     — DM 알림 구독 (토트넘만 / F1만 / 전체)
  /bbdown   — DM 알림 구독 해제
  /bbtest   — DM 테스트 발송
  /bbhelp   — 명령어 전체 안내

================================================================
  Git 이력 확인 방법
================================================================

  ▶ 터미널(cmd/PowerShell)에서 확인
  ----------------------------------
  cd C:\DEV\discord-bot

  # 커밋 목록 간단히 보기
  git log --oneline

  # 커밋 목록 날짜/작성자 포함해서 보기
  git log --pretty=format:"%h %ad %s" --date=short

  # 특정 파일의 변경 이력 보기
  git log --oneline -- bot.py

  # 가장 최근 변경 내용 상세히 보기
  git show HEAD

  ▶ GitHub 웹에서 확인
  ----------------------------------
  https://github.com/cloudland97/BBS 접속
  → [Commits] 탭 클릭 → 전체 커밋 목록
  → 각 커밋 클릭 → 줄 단위 변경 내용 확인
