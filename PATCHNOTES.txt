================================================================
  BBS (BetterBotSpurs) 패치 노트
  작성일: 2026-03-10
  개발 도구: Claude Code + claude-sonnet-4-6
================================================================

[v0.1] 초기 세팅
  - Python 3.14.3 + discord.py 기반 봇 프로젝트 구성
  - 가상환경(venv) 세팅 및 필요 패키지 설치
    (discord.py, aiohttp, icalendar, python-dotenv)
  - run.bat 생성 → 더블클릭으로 봇 실행 가능하도록 구성
  - bss.env에 DISCORD_TOKEN, CHANNEL_ID 등 환경변수 관리

[v0.2] 토트넘 알림 기능
  - /bbup 커맨드: 토트넘 경기 DM 알림 구독
  - /bbdown 커맨드: 구독 해제
  - DM 발송 시점: 경기 24시간 전 / 30분 전 / 10분 전
    (기존 24시간 / 1시간 / 킥오프 → 수정)
  - 30분 전 / 10분 전 DM에 공식 선발 라인업 포함
    (Sofascore API에서 라인업 데이터 가져옴)
  - ICS 캘린더(토트넘 공식) 파싱으로 경기 일정 관리

[v0.3] F1 알림 기능
  - F1 ICS 캘린더 파싱 추가
  - DM에 F1 세션 구분 표시
    예: 프리 프랙티스 1 / 프리 프랙티스 2 / 프리 프랙티스 3
        스프린트 예선 / 스프린트 / 예선 / 본경기
  - /bbf1 커맨드: 다음 GP 전체 세션 일정 한눈에 보기
  - /bbup 구독 모드 선택 기능 추가
    → 토트넘 + F1 전체 / 토트넘만 / F1만

[v0.4] 경기 결과 알림
  - result_loop 추가: 경기 종료 후 자동으로 결과 채널에 전송
  - Sofascore API로 골, 어시스트, 교체 등 주요 이벤트 파싱
  - /bbtime 커맨드: 다음 토트넘 + F1 일정 1개씩 확인

[v0.5] 멀티서버 지원
  - 단일 서버 고정(GUILD_ID) 방식 → 다중 서버 지원으로 변경
  - guild_settings.json으로 서버별 채널 ID 관리
  - /bbset 커맨드: 서버별 알림 채널 지정
  - 글로벌 슬래시 커맨드로 전환 (sync_commands.py로 구 명령어 초기화)

[v0.6] 봇 상태창 추가
  - 봇 상태(presence)에 다음 일정 자동 표시
  - 표시 형식: "vs Atletico 03/11 21:00 | FP1 03/14 09:00"
  - F1은 세션 종류 구분 표시 (프랙티스/예선/본경기)
  - 팀명 자동 단축 처리
    예: Atletico Madrid → Atletico
        Manchester City → Man City 등

[v0.7] /bbhelp 추가
  - /bbhelp 커맨드: 사용 가능한 모든 명령어 한눈에 보기
  - 현재 구독 상태 및 모드도 함께 표시

[v0.8] 코드 구조 리팩터링 (파일 분리)
  - 기존 bot.py 1파일(1000줄+) → 4개 파일로 분리
    config.py  — 환경변수, 상수 모음
    utils.py   — ICS/Sofascore 헬퍼, 포맷 함수
    commands.py — 슬래시 커맨드 전체
    bot.py     — 봇 진입점, 이벤트 루프
  - 이점: 이후 수정 시 관련 파일만 열면 됨 (토큰 절약)

[v0.9] /bblast 추가 및 제거
  - /bblast 커맨드 추가: 최근 경기 결과 + 라인업 + MOM + 선수평점 DM 발송
  - Sofascore API 403 오류로 데이터 수신 불가 확인
  - /bblast 커맨드 제거

================================================================
  현재 커맨드 목록
================================================================
  /bbset    — 알림 채널 지정 (관리자용)
  /bbtime   — 다음 토트넘 + F1 일정 확인
  /bbf1     — F1 다음 GP 전체 세션 일정
  /bbup     — DM 알림 구독 (토트넘만 / F1만 / 전체)
  /bbdown   — DM 알림 구독 해제
  /bbtest   — DM 테스트 발송
  /bbhelp   — 명령어 전체 안내

================================================================
  Git 이력 확인 방법
================================================================

  ▶ 터미널(cmd/PowerShell)에서 확인
  ----------------------------------
  cd C:\DEV\discord-bot

  # 커밋 목록 간단히 보기
  git log --oneline

  # 커밋 목록 날짜/작성자 포함해서 보기
  git log --pretty=format:"%h %ad %s" --date=short

  # 특정 파일의 변경 이력 보기
  git log --oneline -- bot.py

  # 가장 최근 변경 내용 상세히 보기
  git show HEAD

  # 특정 커밋의 변경 내용 보기 (커밋ID는 git log에서 확인)
  git show abc1234

  ▶ GitHub 웹에서 확인
  ----------------------------------
  https://github.com/cloudland97/BBS 접속

  1. [Commits] 탭 클릭 → 전체 커밋 목록
  2. 각 커밋 클릭 → 해당 시점에 어떤 줄이 바뀌었는지 확인
     (초록색 = 추가된 줄, 빨간색 = 삭제된 줄)

  ▶ 현재 변경사항 GitHub에 올리기
  ----------------------------------
  git add .
  git commit -m "v0.9: 파일 분리 및 presence 개선"
  git push
